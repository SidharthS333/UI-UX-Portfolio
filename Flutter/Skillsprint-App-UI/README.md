#  SkillSprint UI

## 📖 Overview

SkillSprint is a Flutter-based learning platform UI that helps users set learning goals and track their progress through an engaging and interactive interface.

The application provides a chat-based experience for entering learning goals and a dedicated progress dashboard to visualize learning journeys.

---

##  UI Features

- Interactive Chat Interface
- Goal Tracking Dashboard
- Circular Progress Indicators
- Character-Based Progress System
- Modern Glassmorphism Design
- Hover Effects
- Smooth Animations
- Dark Theme Interface
- Responsive Flutter UI

---

##  Screens Included

### Chat Screen
Users can enter learning goals through a chatbot-style interface.

### Goals Screen
Displays goals with:
- Progress Percentage
- Character Images
- Animated Goal Cards
- Hover Effects
- Visual Progress Tracking
